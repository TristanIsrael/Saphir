# Package saphir-core

This packages configures the core of Saphir.

## Dependencies

Saphir is built upon **PSEC**, the *Platform Security Enforced Core*.

## Core package

The core package ames at initializing and preparing the hosting system in order to become a sheep-dip station. As it is built upon *PSEC*, it installs different PSEC packages and configures them.

The following operations are achieved on the system configured :
- configuration of the boot manager
- setup of the splash screen
