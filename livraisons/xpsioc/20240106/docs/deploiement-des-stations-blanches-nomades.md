# Déploiement des stations blanches nomades

Une station blanche nomade est une famille de stations blanches regroupant les modèles S (portative avec écran tactile) et les modèles XL nomadisés, c'est-à-dire dont le démarrage ne se fait pas par le réseau.

## Procédure

- Démarrer la station blanche par le réseau en suivant la procédure normale
- Après le démarrage, lorsque la station blanche est prête, appuyer sur les touches Ctrl-Alt-F1
- A l'invite, taper `sudo apk add panoptiscan-nomade`

La nomadisation est déclenchée automatiquement en fonction de la configuration détectée.
