# Approvisionnement

Ce document présente les différentes façon d'approvisionner un nouvel équipement.

## Manuel

L'approvisionnement manuel consiste à suivre une procédure pas-à-pas pour installer le produit en démarrant avec une clé USB Alpine Linux.

1 - Suivre la procédure d'approvisionnement de `PSEC`
  - Les dépôts de Saphir doivent être enregistrés et prêts à être utilisés à l'issue de cette procédure.

2 - Le système doit être démarré sur le noyau XEN

3 - Taper la commande `$ sudo apk add saphir`
